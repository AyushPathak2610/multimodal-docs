# Multimodal ChatBot Documentation

Interactive documentation hub for the Multimodal ChatBot Assistant project.

## Features

- Architecture Guide - System design and component interactions
- Code Walkthrough - Function-by-function explanations
- Interactive UI with search functionality
- Syntax highlighting for code blocks
- Copy-to-clipboard functionality

## Deployment

This site is deployed on Vercel as a static site.

### Deploy to Vercel

**Option 1: Deploy via Vercel CLI (Fastest)**

```bash
# Install Vercel CLI globally
npm install -g vercel

# Deploy (run from project directory)
vercel
```

**Option 2: Deploy via GitHub**

1. Push this repository to GitHub
2. Go to [Vercel Dashboard](https://vercel.com/new)
3. Click "Import Project"
4. Connect your GitHub repository
5. Vercel will auto-detect it as a static site
6. Click "Deploy"

Your site will be live at: `https://your-site-name.vercel.app`

## Local Development

Simply open `index.html` in your browser to view the documentation locally.
